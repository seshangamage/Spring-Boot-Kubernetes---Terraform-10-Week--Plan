Verify Cluster
	kubectl cluster-info
	kubectl get nodes
	kubectl get pods -A

Deploy Your First Pod (nginx)
	kubectl create deployment nginx --image=nginx
	kubectl get deployments
	kubectl get pods

Expose the Deployment (Service)
	kubectl expose deployment nginx --type=NodePort --port=80
	minikube service nginx --url

View Kubernetes Dashboard
	minikube dashboard


Building the docker image from the Springboot code
	docker build -t springboot-microservice:1.0 .

Run the Container Locally - Local url to test http://localhost:8080
	docker run -p 8080:8080 springboot-microservice:1.0

For this project we will not be pushing the images to the docker registry

Run below in a seperate command line opened up with Adminstartor privailges
	minikube start

Apllying Yaml files
	kubectl apply -f .\kubernetes\spingboot-deployment.yaml
	kubectl apply -f .\kubernetes\springboot-service.yaml
